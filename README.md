# Weather Project

Projeto frontend em Vite + TypeScript para exibir informações meteorológicas usando a API Open-Meteo.

## Tecnologias

- Vite
- TypeScript
- CSS
- Open-Meteo API

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

## Build para produção

```bash
npm run build
```

## Deploy no GitHub Pages

O projeto já está configurado para fazer deploy automaticamente via GitHub Actions quando houver push na branch `main`.

### Passos no GitHub

1. Acesse as configurações do repositório.
2. Vá em **Pages**.
3. Em **Source**, selecione:
   - Branch: `gh-pages`
   - Folder: `/root`

O site ficará disponível em:

```text
https://<seu-usuario>.github.io/Project-Weather/
```
